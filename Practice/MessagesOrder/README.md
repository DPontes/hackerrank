# Messages order

[Link to problem description](https://www.hackerrank.com/challenges/messages-order/problem)

To compile and run the program with the input file:

```
> c++ main.cpp -o run.out -std=c++14
> ./run.out < input.txt
```

## Expected output for input.txt

```
Alex
Hello Monique!
What'up?
Not much :(
```
